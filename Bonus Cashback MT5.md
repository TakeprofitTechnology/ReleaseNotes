# Bonus Cashback MT5

## Version 26.06.08.68 (26 June, 2025)
### Features
* Added optional cashback limits per client, with separate caps for positive and negative cashback in Web-GUI rules mode; default behaviour remains unchanged when the mode is disabled.
* Added the MT5 Bonus Cashback plugin to the Web Configurator as a migrated SPA product with full cashback rule management.
* Added optional **Web-GUI cashback rules** configuration, with existing behaviour unchanged by default.
* Added separate cashback percentages for **winning** and **losing** trades.
* Added validation and safe handling of Web-GUI rules files to prevent startup with invalid configurations.
